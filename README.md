Overview
The Campus Lost & Found App helps students and staff report and recover lost or found items on campus. Users can upload item details with images, browse posts, search for specific items, and connect with the rightful owners.
Objectives
Build a simple, user-friendly web application for lost and found items.
Implement CRUD operations along with Searching, Sorting, Filtering, and Pagination.
Add basic admin moderation to remove inappropriate or flagged posts.


Features
Frontend:
Post lost or found items with image, description, location, and category.
Search items by keyword, item name, or location.
Filter items by category (electronics, ID cards, clothing, etc.).
Sort results by date posted, item type, name, or relevance.
Pagination to display items page-wise for better user experience and faster loading.


Backend:
Store user and post data securely in MySQL database.
API support for CRUD operations + Searching + Sorting + Filtering + Pagination.



Tech Stack 
Frontend: HTML,CSS,Javascript,React.js
Backend: Node.js,Express.js
Database: Mongodb


Other Tools : Postman (for API testing), Git & GitHub (for version control), Cloudinary / Firebase / local storage (for storing images)
